Light Through Colored Filters

How to Use This Simulation

This simulation shows how red, green, and blue light pass through colored filters. Each arrow represents one component of light leaving the source.

When light reaches a filter, the filter only allows certain colors to pass through. Any colors that are not transmitted by the filter are absorbed and disappear.

Suggested Procedure

  1. Select the source light (white, yellow, magenta, etc.).
  2. Select one or two filters.
  3. Press Start.
  4. Watch the arrows move toward the filters.

Using the Pause Button

A useful strategy is to press Pause just before the light reaches a filter. This gives you time to predict what will happen next.

Ask yourself:

  • Which colors does this filter allow to pass?
  • Which colors will be absorbed?
  • What color light should remain after the filter?

Then press Resume to see if your prediction is correct.

Try experimenting with different combinations of light sources and filters to see how the colors change.
